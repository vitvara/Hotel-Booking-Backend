## Installation
### Install library
```
pip install -r requirements.txt
```
If you want to use virtual enviroment
```
python -m venv. venv
```

```
`MacOS`
source .venv/bin/activate

`Windows`
.venv/Scripts/activate
```

<sub><sup>! Warning don't name venv dir to .env</sup></sub>

### Setup Database
```
python setup.py <DB url/name>
```
