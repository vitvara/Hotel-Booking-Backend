<h1 align="center"> Hotel-Booking-Project </h1>

## overview
This application was create for Hotel to manage the bookings.

## Requirement
```
python 3.9.7
sqlite3 3.36.0
```

## Installation
Install library
```
pip install -r requirements.txt
```

Setup Database
```
py .\utils\insert_data.py
```
